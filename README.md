# Features

- Flashcard-style UI with front and back cards.
- Automatic card flip after a few seconds.
- Random word selection from a CSV file.
- No repeated words in a session.
- Right / Wrong buttons to track learning.
- Live score tracking.
- Custom card and button images.
- Beginner-friendly, clean Python logic.

## Built With

- Python
- Tkinter (GUI)
- Pandas (CSV handling)
